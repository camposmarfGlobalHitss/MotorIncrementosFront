Acerca del Motor de Incremento

El MIT es una aplicacion que permite generar el calculo masivo de tarifas de clientes que cumplan con unas politicas de incremento de acuerdo a los requerimientos establecidos para cada periodo a incrementar, este se puede hacer para cuentas con CFM como Tarifas de Uso, este proceso desencadena el incremento masivo para los clientes que cumplen todo el proceso
